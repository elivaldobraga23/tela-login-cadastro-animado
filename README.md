# 🔐 Tela de Login e Cadastro Animada

Este projeto é uma **interface moderna de Login e Cadastro com animação**, desenvolvida utilizando **HTML, CSS e JavaScript**.

A aplicação possui uma transição animada entre as telas de **Login** e **Registro**, criando uma experiência visual agradável para o usuário.

O objetivo do projeto é **praticar conceitos de front-end**, como:

* Estruturação de páginas com HTML
* Estilização avançada com CSS
* Animações e transições
* Manipulação de classes com JavaScript
* Criação de interfaces modernas

1️⃣ Clone o repositório


